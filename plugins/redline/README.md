# redline

A Claude Code plugin that feeds the agent your **manual working-tree edits made
between its turns**, as a `git diff` — so the next prompt carries exactly what
you changed into Claude's context. An optional status line shows those pending
edits live in the status bar.

Full documentation — install and how it works —
lives in the [repository README](https://github.com/michail-nikolaev/redline#readme).
